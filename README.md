# Projeto – Sistema de Missões Lunares

## Como executar
1. Instale Java 17+
2. Instale Maven
3. Execute:
mvn clean package
mvn exec:java -Dexec.mainClass="br.edu.lunar.view.ConsoleApp"


## Tecnologias
- Java 17  
- Maven  
- NitriteDB  
- Serialização binária  

## Funcionalidades
- Criar missões  
- Listar missões  
- Registrar astronautas  
- Registrar resultados científicos  
- Persistência dupla
